Para plantillas:
	-->Se crea un objeto Template 
		plt = Template(doc_externo.read())
	-->Se crea el contexto si tiene contenido dinamico
		ctxt = Context()
	--> rederizar el objeto Template
		documento = plt.render(ctxt)

En una vista se haria asi
	//Importar from django.template import Template,Context
	doc_externo = open(path al documento html)
	plt = Template(doc_externo.read())
	doc_externo.close()
	ctx = Context() //si esta vacio significa que no tiene contenido dinamico
	documento = plt.render(ctx)
	return Httpresponse(documento)
